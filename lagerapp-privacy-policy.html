Sekretesspolicy - Lagerapp

SpaceN00b Apps

Senast uppdaterad: 19 november 2025

1. Introduktion

Denna sekretesspolicy gäller för mobilapplikationen Lagerapp ("vi", "vår", "appen"). Policyn beskriver hur vi samlar in, använder, lagrar och skyddar dina personuppgifter när du använder Lagerapp.

Genom att använda appen godkänner du de metoder för insamling och användning av information som beskrivs i denna policy.

2. Personuppgiftsansvarig

Personuppgiftsansvarig för behandlingen av personuppgifter i Lagerapp är SpaceN00b Apps. Kontaktinformation finns i avsnitt 12.

3. Vilka uppgifter samlar vi in?

3.1 Kontoinformation

När du registrerar dig och använder appen samlar vi in:

  • E-postadress (används som användarnamn)
  • Lösenord (krypterat)
  • Fullständigt namn
  • Telefonnummer
  • Kontraktsinformation

3.2 Kameraåtkomst

Appen använder din enhets kamera för att:

  • Skanna streckkoder på produkter och artiklar
  • Underlätta snabb artikelsökning genom streckkodsskanning

Viktigt: Kameran används endast för streckkodsskanning. Inga bilder sparas, lagras eller delas med någon tredje part. Kameran aktiveras endast när du väljer att använda skanningsfunktionen.

3.3 Arbetsdokument och data

All data du skapar i appen lagras, inklusive:

  • Rekvisitioner (lageruttag)
  • Varukorgsinformation
  • Artikelinformation (artikelnummer, namn, beskrivningar)
  • Satsinformation (fördefinierade artikelgrupper)
  • Mottagarinformation
  • Uttagshistorik
  • PDF-dokument för rekvisitioner

3.4 Teknisk information

För att förbättra appens prestanda och stabilitet kan vi samla in:

  • Enhetstyp och operativsystem
  • Appversion
  • Kraschrapporter och felloggar

4. Hur använder vi dina uppgifter?

Vi använder insamlade uppgifter för följande ändamål:

  • Tillhandahålla tjänsten: För att du ska kunna använda appens funktioner för lagerhantering
  • Autentisering: För att säkerställa att endast behöriga användare får tillgång
  • Dokumentation: För att skapa, lagra och dela rekvisitioner och lageruttag
  • Streckkodsskanning: För att snabbt identifiera artiklar genom kameran
  • Historik: För att spåra tidigare lageruttag och rekvisitioner
  • Förbättring: För att analysera användning och förbättra appens funktionalitet
  • Felsökning: För att identifiera och åtgärda tekniska problem
  • Säkerhet: För att skydda mot missbruk och säkerhetshot

5. Hur lagras dina uppgifter?

5.1 Lokal lagring på enhet

Viss data lagras lokalt på din enhet för att förbättra prestanda:

  • Inloggningstoken (för automatisk inloggning)
  • Användarinställningar
  • Tillfällig cache för artikeldata

5.2 Server-baserad lagring

Följande data lagras på säkra servrar:

  • Kontoinformation
  • Alla skapade rekvisitioner och lageruttag
  • Artiklar och satser
  • Mottagare
  • Användarprofiler
  • PDF-dokument

All data överförs krypterat via HTTPS och lagras på servrar inom EU (Sverige).

6. Delning med tredje part

Vi delar INTE dina personuppgifter med några tredje parter.

Appen är byggd med Expo/React Native-ramverket, men ingen personlig data skickas till Expo eller andra externa tjänster. All data lagras på våra egna servrar.

Vi säljer aldrig dina personuppgifter till tredje part.

7. Dataskydd och säkerhet

Vi vidtar följande säkerhetsåtgärder för att skydda dina uppgifter:

  • Krypterad dataöverföring (HTTPS/TLS)
  • Säker autentisering med JWT-tokens
  • Krypterade lösenord
  • Rollbaserad åtkomstkontroll (användare, verktyg, admin)
  • Regelbundna säkerhetsuppdateringar
  • Automatiska säkerhetskopior av databasen
  • Åtkomstkontroll baserad på kontrakt (multi-tenant isolation)

Trots dessa åtgärder kan ingen metod för dataöverföring eller elektronisk lagring vara 100% säker. Vi kan därför inte garantera absolut säkerhet.

8. Dina rättigheter (GDPR)

Enligt GDPR har du följande rättigheter:

  • Rätt till tillgång: Du kan begära en kopia av dina personuppgifter
  • Rätt till rättelse: Du kan uppdatera felaktig eller ofullständig information
  • Rätt till radering: Du kan begära att vi raderar dina personuppgifter
  • Rätt till begränsning: Du kan begära att vi begränsar behandlingen
  • Rätt till dataportabilitet: Du kan få dina data i ett maskinläsbart format (CSV)
  • Rätt att göra invändningar: Du kan invända mot vår behandling av dina uppgifter

För att utöva dina rättigheter, kontakta oss på den e-postadress som anges i kontaktinformation nedan.

9. Barn och personuppgifter

Lagerapp är avsedd för professionellt bruk inom lagerhantering och är inte riktad till personer under 18 år. Vi samlar inte medvetet in personuppgifter från barn.

10. Datalagring och radering

Vi lagrar dina personuppgifter så länge du har ett aktivt konto. Om du begär radering av ditt konto kommer dina personuppgifter att raderas inom 30 dagar.

Vissa uppgifter kan behöva behållas längre för att uppfylla juridiska krav eller arkiveringskrav som gäller för arbetsdokumentation och lagerhantering.

11. Ändringar i sekretesspolicyn

Vi kan uppdatera denna sekretesspolicy från tid till annan. Vi kommer att meddela dig om väsentliga ändringar genom ett meddelande i appen eller via e-post.

Vi rekommenderar att du regelbundet granskar denna policy för att hålla dig informerad om hur vi skyddar dina uppgifter.
